## Setup & Run

### Step 1 — Run your analytics pipeline first
```bash
python src/main.py
```
This generates `output/customer_loyalty_report.csv`

### Step 2 — Install dashboard dependencies
```bash
pip install flask flask-cors pandas requests
```

### Step 3 — Start Ollama (in a separate terminal)
```bash
ollama serve
```
Make sure llama3 is pulled:
```bash
ollama pull llama3
```

### Step 4 — Start the Flask dashboard server
```bash
python src/analytics_server.py
```

### Step 5 — Open dashboard
Visit: **http://localhost:5000**

## Features

| Feature | Description |
|---|---|
| KPI Cards | Total customers, revenue, active count, churn risk count |
| Billing Table | Searchable, filterable customer records |
| Customer Modal | Click any row → full detail popup with coupon |
| AI Chatbot | Ask about any customer by name/ID |
| Quick Prompts | One-click common questions |
| Ask AI from Modal | Click button in modal → auto-asks AI about that customer |

## Example Chat Queries
- "Give me a full report on Rahul Sharma"
- "Who are the high churn risk customers?"
- "Which customers should be upsold to Platinum?"
- "Give me an overall summary"
